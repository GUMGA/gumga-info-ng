Descrição

O componente gumga-info-ng é um alerta informativo em modal para avisos antes de operações importantes.

Instalação

// npm
npm i gumga-info-ng --save

// bower
bower install gumga-info-ng --save
Usando
  • HTML
  • JavaScript
<button type="button" ng-click="example()"
    gumga-info="Informação importante"
    class="btn btn-info"
    dismiss-button="OK"
    dismiss-button-class="btn btn-default">
    Informar
</button>
angular.module('app', ['ui.bootstrap','gumga.info'])
  .controller('Ctrl', function () {
    var ctrl = this
    ctrl.example = function() {
      alert('Função executada')
    }
  })
